# Music-Bot
A Discord.js Music Bot

### How To Setup ?


1. First Do `git clone https://github.com/BotStudios/Music-Bot.git` In Your Terminal


2. After Clonning The Repo, Do `cd Music-Bot`


3. Then `npm i`


4. Replace YOUR_TOKEN To Your Bot's Token In index.js


5. Done
