# Music-Bot
A Discord.js Music Bot

### How To Setup ?


1. First Do `git clone https://github.com/BotStudios/Music-Bot.git` In Your Terminal


2. Then `npm i`


3. Replace YOUR_TOKEN To Your Bot's Token In index.js


4. Done
